USAGE:
    lhabia <target> [options]

TARGETS:
    IP Address
    Domain Name
    CIDR Range

SCAN PROFILES:
    -q              Quick Scan
    -F              Full Scan
    -R              Recon Mode
    -WEB            Web Audit
    -ALL            Complete Assessment

SCAN TECHNIQUES:
    -sS             SYN Scan
    -sT             TCP Connect
    -sU             UDP Scan
    -sV             Service Detection
    -sB             Banner Grab

PORTS:
    -p <ports>      Specific Ports
    -r <range>      Port Range
    -t100           Top 100 Ports
    -t1000          Top 1000 Ports

HOST DISCOVERY:
    -P              Ping Scan
    -HD             Host Discovery
    -PN             Skip Discovery

WEB ENUMERATION:
    --web           Web Fingerprint
    --tls           SSL/TLS Analysis
    --headers       Security Headers
    --tech          Technology Detection

NETWORK ENUMERATION:
    --dns           DNS Enumeration
    --whois         WHOIS Lookup
    --geo           GeoIP Lookup
    --trace         Traceroute
    --os            OS Detection

REPORTING:
    -o <file>       Save Text Report
    -O <file>       Save JSON Report
    --html          Generate HTML Report

PERFORMANCE:
    -T <threads>    Thread Count
    -W <timeout>    Connection Timeout

OUTPUT:
    -v              Verbose
    -z              Silent

INFORMATION:
    -h              Help
    -V              Version

EXAMPLES:

    lhabia 192.168.1.1 -q

    lhabia scanme.nmap.org -R

    lhabia target.com -WEB

    lhabia target.com -F -sV --html

    lhabia target.com --dns --whois

    lhabia 10.10.10.10 -ALL
